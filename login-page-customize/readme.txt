== About Login Page Customize ==

**Login Page Customize** is a very useful to customize a WordPress login page plugin. Login Page Customize plugin has more functions with all things of login page and register page. You can change background image, logo and primany, secondary color. login page all text change option and Register page all text change option. You can choose the link open options in same tab or a new tab.
Everybody can use this plugin on their websites. If you want to customize the WordPress register and login page, you can use this plugin. It is best to customize WordPress login page of a WP website. "Login Page Customize" is easy to use.

== How to use ==
You can use this plugin with some steps, like:

* Search and Install the Login Page Customize Plugin. You will be redirected to the plugin admin page after installing it.
* You can change everything from here like change background image, logo change, all links change, text change, login and total register page customize.
*  Then you go to your website login page and you will see the awesome login page with all functionality.
